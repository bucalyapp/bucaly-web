# Bucaly Web

Proyecto limpio, listo para publicar desde cero.

## Contenido
- index.html → la app completa con el triaje, ya con el nombre Bucaly
- manifest.json → hace que sea instalable como app
- sw.js → funcionamiento sin internet
- icon-192.png / icon-512.png → íconos con la "B"
- vercel.json → configuración de publicación

## Pasos (ver guía completa en el chat)
1. git init
2. git add .
3. git commit -m "Bucaly version inicial"
4. Crear repo nuevo en GitHub llamado bucaly-web
5. git remote add origin (la URL que te da GitHub)
6. git branch -M main
7. git push -u origin main
8. Importar el repo en Vercel y hacer Deploy
